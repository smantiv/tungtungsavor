✅ ¿Qué es una rama?

Una rama (branch) es como una copia del proyecto donde puedes trabajar sin dañar la principal (main).

Ejemplo:

👉 main = versión oficial
👉 menu-css = tu versión para probar cosas

Luego se juntan (merge).

🚀 PASO A PASO: Crear una rama

Abre la terminal en tu proyecto.

📌 1️⃣ Mira en qué rama estás

bash
git branch

La que tiene * es la actual.

Ejemplo:

css
* main

📌 2️⃣ Crea una nueva rama

Ejemplo para tu parte:

bash
git branch menu-css

👉 Esto crea la rama, pero NO entras en ella todavía.

📌 3️⃣ Cambia a esa rama

bash
git checkout menu-css

O en una sola línea:

bash
git checkout -b menu-css

(la crea y entra al tiempo ✅)

📌 4️⃣ Verifica

bash
git branch

Debe salir:

css
* menu-css
  main

✅ Ya estás trabajando en tu rama

Ahora todo lo que hagas:

✔ Cambios
✔ Commits

Solo quedan en menu-css.

📤 PASO 5: Sube tu rama a GitHub

Para que los demás la vean:

bash
git push -u origin menu-css

Solo la primera vez.

Después:

bash
git push
